DESKRIPSI WEBSITE
Website ini merupakan sistem pemesanan lapangan olahraga berbasis web yang dirancang untuk memudahkan pengguna dalam melakukan penyewaan lapangan futsal dan voli secara online.
Website menyediakan informasi lengkap mengenai:
-Profil penyedia lapangan
-Fasilitas yang tersedia
-Daftar harga sewa
-Proses booking lapangan secara bertahap
Pengguna dapat melakukan pemesanan dengan mengisi biodata, memilih jenis lapangan, menentukan tanggal dan waktu, mengatur durasi sewa, hingga melihat ringkasan pemesanan.
Website ini dibangun menggunakan teknologi HTML5, CSS3, dan JavaScript dengan struktur folder yang rapi sehingga mudah dikembangkan dan dipelihara.

DAFTAR FITUR
1.Form Booking Lapangan Bertahap
  Pengguna melakukan pemesanan melalui beberapa halaman mulai dari biodata, pemilihan lapangan, penentuan tanggal & jam,        durasi, hingga ringkasan pemesanan.
2.Validasi Form Biodata (JavaScript)
  Sistem memeriksa:
  Format email
  Panjang nomor telepon
  Field wajib tidak boleh kosong
3.Dark Mode Toggle
  Pengguna dapat mengaktifkan atau menonaktifkan mode gelap menggunakan tombol toggle.
4.Navigasi Dinamis Antar Halaman
  Navigasi antar halaman menggunakan event klik JavaScript tanpa reload berlebihan.
5.Update Jadwal Otomatis
  Jadwal lapangan akan berubah otomatis menjadi “Booked” setelah pengguna melakukan pemesanan.
6.Ringkasan Booking Otomatis
  Data yang dimasukkan pengguna akan langsung ditampilkan pada halaman ringkasan pemesanan.
7.Manipulasi DOM Interaktif
  JavaScript digunakan untuk mengubah isi teks, class, dan tampilan elemen berdasarkan input pengguna.
8.Media Galeri Lapangan
  Website menampilkan gambar lapangan futsal dan voli sebagai media pendukung.

TEKNOLOGI YANG DIGUNAKAN

1.HTML5
  Digunakan untuk membangun struktur halaman website seperti halaman utama, halaman informasi, serta form pemesanan lapangan.
2.CSS3
  Digunakan untuk mengatur tampilan website, termasuk layout, warna, font, dan desain agar tampilan konsisten di semua          halaman.
3.JavaScript (Vanilla JS)
  Digunakan untuk membuat fitur interaktif seperti validasi form, dark mode, navigasi dinamis, update jadwal otomatis, dan      ringkasan pemesanan.
4.Media Gambar (JPEG / PNG)
  Digunakan untuk menampilkan foto lapangan futsal dan voli sebagai konten visual website.
5.Git & GitHub
  Digunakan sebagai version control dan media pengumpulan project.
